Latency Phase Animation Options

Option 1: Ping-Pong Dot

Dot bounces between You and Server. Each trip shows a latency reading.

Option 2: Multi-Server Ping Race

Three server candidates race. Fastest wins selection.

Option 3: Radar / Sonar Pulse

Ring pulses out and bounces back. Diameter = latency.

Option 4: Packet Hop Trail

Dot hops through network path, bounces back with trail.
